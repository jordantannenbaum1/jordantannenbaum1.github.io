About this portfolio

Data work in public.

I believe data should be reproducible, transparent, and opinionated. Every project here shows its work — raw sources, cleaning logic, transformation code, and the design decisions behind the visualizations.

My professional focus is data infrastructure and analytics for mission-driven organizations. My personal interests in housing equity, income inequality, and wealth distribution show up in the projects I choose to build here.

This site is a GitHub Pages deployment from a single repository. No build step. Each project is a self-contained folder with an index.html and associated assets. The charts are interactive D3.js — no static PNGs, no external dashboarding tools.
