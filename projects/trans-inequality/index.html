What it is
A multi-dimensional look at inequality faced by transgender Americans, with specific focus on trans women of color who face compounding disparities in life expectancy, income, housing stability, and violence. The data exists. It just rarely gets modeled together.
Data Sources
US Trans Survey (USTS) — income, housing, employment
CDC NCHHSTP — health outcomes, violence data

-- int_lifespan_est.sql (abridged life table method)
with homicide_rates as (
    select
        race_category, gender_identity, age_group,
        count(*) as homicide_count,
        pop.estimated_population,
        round(count(*) * 100000.0
            / nullif(pop.estimated_population, 0), 2)
            as rate_per_100k
    from {{ ref('stg_homicides') }} h
    left join {{ ref('stg_population_est') }} pop
        using (race_category, gender_identity, age_group)
    group by 1,2,3,5
)
select
    race_category, gender_identity, age_group,
    rate_per_100k / 100000            as mx,
    1 - exp(-rate_per_100k / 100000) as qx
from homicide_rates
